# **App Name**: Ida Olive Cottage Retreat

## Core Features:

- Cottage Showcase: Display cottage details (description, amenities, photos) sourced from Lekkeslaap, enhanced by AI suggestions based on user reviews.
- Things to Do: Present nearby attractions and activities (hiking, wine tasting, stargazing) with dynamic content updates via the admin portal.
- Availability Calendar: Display an availability calendar, sourced from Nightsbridge. Includes links to rates and detailed room information.
- Enquiry Form: Offer a dynamic enquiry form capturing name, email, phone, dates, and guest count; submissions are emailed and stored in the admin panel.
- Links and Location: Provide social media links and an embedded Google Maps view of the cottage location.
- AI Review Analysis: AI-powered tool to analyze and respond to guest reviews, incorporating key themes into promotional material and website copy, which helps to enhance content in various sections of the website (e.g., things to do).

## Style Guidelines:

- Primary color: Olive green (#808000) for a natural, earthy feel.
- Background color: Light tan (#F5F5DC), a desaturated version of olive green.
- Accent color: Dark orange (#A0522D), for highlights and call to action buttons.
- Body and headline font: 'Aptos' (sans-serif) for a modern look, with 'Footlight MT Light' (serif) for headings to evoke warmth and approachability. Note: currently only Google Fonts are supported.
- Use nature-inspired icons (leaves, animals, stars) in a minimalist style.
- Employ a single-page design with clear sections for different topics and a quick-link navigation.
- Subtle animations like fading transitions for loading content, giving a smooth, calm user experience.
